The Budding Techie 2021-22 is a competition to showcase the web development skills of students from
Isha Vidhya Schools in Coimbatore, Erode, Dharmapuri, Vanavasi, Karur, Thuthukudi, Cuddalore, Villupuram, Nagercoil and Aragonda.

This competition is being held as part of the completion of the courses on HTML, CSS and JS that the students underwent in vidhya.io (and elsewhere) during the 2021-22 academic year.

1st Prize - Smatch Watch worth Rs. 5500

2nd Prize - Amazon Echo Dot worth Rs. 3250

3rd Prize - Boat TWS Earbuds worth Rs. 1299

On top of this, all participants get a participation prize!

Web Development Competition details

Deadline - 31st March 2022

Submission Guidelines

  • You must create a website using what you have learned so far.
  • Deploy your website files to Netlify and submit a project link. Watch this video to learn how to deploy site to Netlify. You'll need to be on a computer for this.
  • For 1st, 2nd and 3rd Prizes - Submit your project link before deadline. Whoever scores the highest number of points for the projects will win. See the project evaluation criteria below to see how points will be awarded for the projects.
  • For course participation prize - Submit a link to a valid project link by the deadline.

Project Topics

You have to create a website as your final project. You can pick one of the following options or choose your own kind of website to build:-

  1. Website for a school
  2. Website for a photographer
  3. Website for a shop keeper
  4. Website to show tourist places in your region
  5. Website for your team
  6. ***Your own Idea!***

Project evaluation criteria

After you submit your project, it will be graded and given points (out of a total of 60 points) based on the following criteria:-

  1. Overall presentation (10 points) - How good the project feels on the whole, if it feels cohesive with proper design choices.
  2. Complexity of features (10 points) - No. of pages in your website, how much content, how many varieties of features
  3. Creativity (10 points) - How realistic your website looks, use of media (images/videos/audio clips/animations)
  4. Learning to learn (10 points) - Bonus features (use of features not taught in the course)
  5. Originality (10 points) - How unique is your project?
  6. Code quality (10 points) - Formatting of code, no repetitive code, fewer lines of code, proper comments

The project with the maximum points will win the competition. In case of a tie, the winner will be decided based on the score on the leaderboard app.
